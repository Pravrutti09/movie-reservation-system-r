# Movie Reservation System (R)

This project is a simple Movie Reservation System developed using the R programming language. The goal of the system is to simulate a basic movie theater seat booking process.

The application allows users to view the seating arrangement and reserve seats for available movies through a menu-driven console interface.

## Features

View available movies  
Display theater seating arrangement  
Reserve seats for selected movies  
Prevent booking of already reserved seats  
Simple console-based interface  

## Technologies Used

R Programming  
Vectors and Matrices  
Basic Data Management  

## How to Run

1. Open the script in RStudio or any R environment.
2. Run the program.
3. Follow the menu options to view seats or reserve a seat.

## Purpose of the Project

This project was created to practice programming concepts in R and understand how data structures like vectors and matrices can be used to build simple management systems.
